# NFC AI Assistant

AI-powered document question-answering assistant built during internship work at Nuclear Fuel Complex (NFC).

## Features
- PDF document processing
- AI-based question answering
- Vector database search
- Semantic document retrieval

## Technologies Used
- Python
- Gemini API
- LangChain
- Vector Embeddings

## Project Structure
- app.py → Main application
- create_vector_store.py → Creates embeddings/vector store
- documents/ → PDF and text resources

## Future Improvements
- Streamlit interface
- Offline deployment
- Multi-document support
